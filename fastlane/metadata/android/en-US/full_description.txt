<p><i>Expandroid</i> lets you send custom messages with a trigger. Want to quickly type out the current date in a specific format? Or do you want your emojis to replace your triggers? You can do it all cross platform with espanso but now on android too with this app!</p><p><br><b>Features supported:</b></p><ul><li>Text Expander feature with espanso configuration support</li><li>Compare two text sequences</li><li>Replace certain values in text</li><li>Check if your text contains a certain value</li><li>Convert to upper case or lower case</li></ul><p>Ignoring case or otherwise is supported for all options. This PRO version does not contain any ads and all features are unlocked by default.</p><p><br><b>This app uses the Accessibility API!</b></p><p>This app uses Accessibility API to detect whether a saved shortcut is typed and replace it with a corresponding phrase.</p>
